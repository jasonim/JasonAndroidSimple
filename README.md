# Android Practices
本项目是Android Practices， 虽然不是现在不是Best Practices，但是我相信以后是； 'you will be best practices!'
面对现在众多第三方库，很短时间内很难知道哪一个是最好的，最开始时主要为了学习使用， 后来慢慢发现还可以让更多人知道整个过程， 而不是仅仅在模仿轮子，我们也可以改造轮子。
